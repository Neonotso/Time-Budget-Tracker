# Project Tracker

## How to Use This Tracker
For each project, fill out:
- **Project Name**: Clear, specific title
- **Done Criteria**: What does "finished" actually look like? Be specific.
- **Next Action**: The very next physical thing you need to do (call, buy, write, measure, etc.)
- **Time Estimate**: Rough guess (15 min, 2 hr, 1 day, etc.)
- **Due/Target**: When would you like it done by? (flexible is ok)
- **Energy Level**: High/Med/Low - when you're best suited to work on it
- **Status**: Not Started / In Progress / Blocked / Done
